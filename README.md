# DressHub APIs

Basic PHP API structure for DressHub ecommerce application.

## Setup

1. **Database**: Create a MySQL database named `ecommerce_app` in localhost
2. **Config**: Database credentials are set in `config/Database.php`
3. **Access**: Visit `http://localhost/DressHub%20APIs/`

## Project Structure

```
├── config/
│   └── Database.php        # Database connection
├── models/
│   └── Product.php         # Product model
├── controllers/
│   └── ProductController.php # Product controller
└── index.php              # Main entry point
```

## Database Connection

- **Host**: localhost
- **Database**: ecommerce_app
- **Username**: root
- **Password**: (empty by default in XAMPP)

You can update credentials in `config/Database.php` if needed.
